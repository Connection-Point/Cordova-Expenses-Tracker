This is Cordova Project bootstrapped with cordova create

Getting Started

First install the necessary packages and dependency.

"npm i"

Then, add your preferred cordova platform
"cordova platform add <platfrom_name>"

Open database.js (./www/js/database.js) using your preferred code editor.


Change Line 6 To Your Server URL.
Example: const serverURL = 'https://192.168.0.109/COMP3402/';

To run, type this in the terminal.
"cordova run <platform_name>"
